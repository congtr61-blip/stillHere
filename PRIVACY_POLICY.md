## 概览

欢迎使用 StillHere（以下简称"本应用"）。我们重视您的隐私，并致力于透明地说明我们如何收集、使用和保护您的个人信息。

**简单来说：**
- 我们仅收集为您提供服务所需的个人信息
- 我们不会将您的数据出售给第三方
- 您可以随时请求访问、修改或删除您的数据
- 我们使用行业标准的加密和安全措施保护您的数据

## 我们收集的数据

### 1. 认证信息

#### 手机号码
- **是什么：** 用于用户注册和登录的电话号码
- **为什么收集：** 
  - 进行用户身份验证
  - 发送 SMS 验证码

- **存储位置：** Firestore 数据库
- **加密状态：** ✅ 加密存储

- **是什么：** 用户提供的电子邮件地址
- **为什么收集：**
  - 账户注册和登录
  - 邮箱验证和确认
  - 账户安全通知
  - 账户恢复
- **存储位置：** Firestore 数据库和 Firebase Authentication
- **加密状态：** ✅ 加密存储

#### Google 账户信息
- **是什么：** 通过 Google 登录时收集的信息

  - 邮箱地址
  - 姓名（如果提供）
  - 个人资料照片（如果提供）
- **为什么收集：**

## 数据保护

### 加密

**所有传输：** ✅ TLS/SSL 加密
- 所有网络通信都通过 HTTPS 加密
- 数据在服务器之间传输时被加密

**静态存储：** ✅ 数据库加密
- Firebase 自动加密存储在 Firestore 中的数据
- 敏感信息（如密码）使用行业标准加密

### 访问控制

**身份验证：**
- Firebase Authentication 处理所有用户认证
- 只有认证用户可以访问自己的数据
- 服务器端验证所有请求

**授权规则：**
```javascript
// Firestore 安全规则示例：
match /users/{userId} {
  allow read, write: if request.auth.uid == userId;
}

match /contacts/{userId} {
  allow read, write: if request.auth.uid == userId;
}
```

这意味着：
- 用户只能访问自己的数据
- 其他用户无法看到您的信息
- 管理员无法读取您的数据

### 安全措施

✅ **已实施的安全措施：**
- 定期安全审计
- 防火墙和 DDoS 保护（Google Cloud 提供）
- 加密密钥管理
- 日志和监控
- 安全的密码处理

### 数据备份

- Firebase 自动备份所有数据
- 备份也被加密
- 备份存储在安全的地理位置

## 数据保留

### 我们保留您的数据多长时间？

#### 活跃账户期间
- 所有账户数据在您使用账户时被保留
- 这包括手机号、邮箱、验证状态等

#### 账户删除后
- **立即删除：**
  - 所有个人识别信息
  - 联系方式
  - 验证记录

- **保留 30 天（用于恢复）：**
  - 账户数据（以防您需要恢复账户）
  - 30 天后永久删除

- **保留期限为法律要求的最长期限：**
  - 交易日志（税务和法律要求，通常 7 年）
  - 安全审计日志（1 年）
  - 法律诉讼相关数据（直到诉讼解决）

#### 被动账户
- 如果您的账户 12 个月内无活动，我们会：
  - 发送提醒邮件
  - 给予 30 天继续使用的机会
  - 30 天后，您的账户将被视为不活跃
  - 不活跃账户的数据在 12 个月后删除

### 数据最小化

我们遵循"数据最小化"原则：
- 仅收集为提供服务所必须的数据
- 定期审查和删除不必要的数据
- 不保留"以备将来使用"的数据

### 我们对儿童的政策

✋ **StillHere 应用不适合 13 岁以下的儿童。**

**关键点：**
- 我们不故意收集 13 岁以下儿童的个人信息

### 13-18 岁青少年

- 我们遵守适用的青少年隐私法
- 我们建议父母监督青少年账户
- 青少年拥有相同的隐私权

## 快速参考指南

### 常见问题

**Q: 您会将我的手机号出售给市场营销公司吗？**
A: 不会。我们从不出售任何个人数据。

**Q: 如果我删除我的账户会发生什么？**
A: 您的所有个人数据将被删除。只有合法要求的数据（如税务记录）会根据法律要求保留。

**Q: 您是否追踪我的位置？**
A: 不会。我们收集 IP 地址用于安全目的，但不追踪您的物理位置。

**Q: 我如何更新我的信息？**
A: 在应用中选择"编辑资料"或"账户设置"，然后修改您的信息。

**Q: 我如何知道我的数据是否被泄露？**
A: 如果发生安全事件，我们会在 72 小时内通过电子邮件通知您。

**Q: 您使用什么加密？**
A: 
- 传输：TLS/SSL（HTTPS）
- 存储：AES-256（Firebase 标准加密）
- 备份：同样的加密标准

**Q: 第三方 (如 Google) 可以看到我的数据吗？**
A: 只有他们为了提供服务所必需的信息。所有敏感数据都被加密。
