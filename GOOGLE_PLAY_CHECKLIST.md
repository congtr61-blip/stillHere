## 🚀 发布步骤

### 第 1 步：准备签名（如果未完成）

```bash
# 生成密钥存储文件（Android）
keytool -genkey -v -keystore ~/key.jks \
  -keyalg RSA -keysize 2048 -validity 10000 \
  -alias key

# 将密钥存储在安全的地方
# ⚠️ 非常重要：丢失密钥意味着无法更新应用！
```

### 第 2 步：构建发布版 APK/AAB

```bash
# 构建 AAB（推荐）
flutter build appbundle --release

# 输出位置：build/app/outputs/bundle/release/app-release.aab
```

### 第 3 步：在 Google Play Console 中上传

1. 访问 https://play.google.com/console
2. 选择应用
3. 点击"发布"→"Production Release"
4. 上传 app-release.aab
5. 填写发布说明
6. 提交审核

### 第 4 步：等待审核

- 初审通常 1-2 小时
- 完整审核通常 24-48 小时
- 如被拒，查看具体原因并修改
