## 🐛 遇到的问题和解决方案

### 问题 1：reCAPTCHA 错误

```
错误：reCAPTCHA token response is invalid
解决：
1. 检查 main.dart 中 reCAPTCHA 配置
2. 在 debug 模式下自动禁用
3. 如需测试，使用虚拟号：+8611111111111
```

### 问题 2：Firebase 初始化失败

```
错误：No Firebase App '[DEFAULT]' has been created
解决：
1. 确认 firebase.json 配置正确
2. 确认 google-services.json（Android）或 GoogleService-Info.plist（iOS）
3. 网页版无需这些文件，使用 Firebase config
```

### 问题 3：Google Sign-in 不工作

```
错误：signIn method is discouraged
解决：
1. 这是已知警告，不影响功能
2. 按照警告提示迁移到 renderButton（未来）
```

### 问题 4：邮件未收到

```
原因：邮箱验证邮件延迟或进入垃圾箱
解决：
1. 检查垃圾邮件文件夹
2. 回到应用点击"重新发送验证"
3. 等待 1-5 分钟
```
