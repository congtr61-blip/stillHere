## 🎬 使用流程

### 用户角度

```
场景 A: 绑定新邮箱（简单 ✅）
1. 打开 → 联系方式管理
2. 输入新邮箱：new@gmail.com
3. 点击"绑定邮箱"
4. ✅ 系统提示：邮件已发送，请验证
5. 打开邮件，点击验证链接
6. ✅ 完成！邮箱已验证

时间：1-5 分钟
```

```
场景 B: 链接已使用邮箱（认证 ✅）
1. 打开 → 联系方式管理
2. 输入已使用邮箱：user@qq.com
3. 点击"绑定邮箱"
4. 系统提示：邮箱已被使用，显示链接对话框
5. 点击"使用密码链接"
6. 输入该邮箱的密码
7. 点击"链接账户"
8. ✅ 完成！账户已链接

时间：3-5 分钟
```

```
场景 C: 自动识别合并（最佳 ✅）
1. 现在用 Google 登录（账户 A）
2. 打开 → 联系方式管理
3. 输入 user@qq.com（曾用密码登录的账户 B）
4. 系统检测冲突 → 提示"邮箱已被使用"
5. （后续）如果用 user@qq.com + 密码重新登录
6. ✅ 系统自动识别并链接两个账户

时间：2-3 分钟（跨登录）
```

## 🔒 安全谈话

✅ **密码安全**
- 密码仅用一次验证
- 不会被保存或记录
- 验证完立即销毁

✅ **数据隐私**
- 用户无法访问他人数据
- Firestore 规则严格限制
- 链接账户时无数据合并（用户数据保持独立）

✅ **邮箱验证**
- 验证链接唯一且一次性
- 24 小时有效期
- 支持重新发送

✅ **金钥管理**
- Firebase 凭证系统管理
- 完全由 Google 处理加密
- 没有密钥被暴露的风险
